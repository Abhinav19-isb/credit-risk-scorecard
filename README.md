# Credit Risk Scorecard Builder 🏦

An LLM Skill Package that instructs an AI to perform end-to-end credit risk analysis —
from raw data → feature engineering → dual-model scorecard → professional HTML report.

## Domain
Financial Analytics — Credit Risk

## Dataset
UCI Credit Card Default Dataset (30,000 customers, 23 features)

## Pipeline Stages
1. Data Validation & Profiling
2. Feature Engineering (WoE, IV, payment behaviour features)
3. Modelling — Logistic Regression Scorecard vs XGBoost
4. Model Validation — KS Statistic, Gini, AUC-ROC
5. Insight Generation & Score Interpretation
6. Professional HTML Report Generation
